Canvas API POC

The Canvas API exposes a 2D drawing surface accessed via a CanvasRenderingContext2D. You issue imperative drawing commands (fillRect, arc, drawImage) — no virtual DOM, no React. Perfect for games, data viz, image editors.

1. Shapes and paths

Basic primitives: rectangles, circles, lines, gradients.

2. Animation with requestAnimationFrame

Bouncing balls — the canvas redraws ~60 times per second.

3. Interactive drawing

Click and drag to draw. Captures pointer events on the canvas.
