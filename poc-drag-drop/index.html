Drag and Drop API POC

The HTML5 Drag and Drop API has a quirky-but-powerful event model. You mark elements with draggable="true", then listen for dragstart, dragover, drop, etc. The dataTransfer object carries data between the source and target.

1. Sortable list

Drag items to reorder. State of the list is reflected in the data attributes.
