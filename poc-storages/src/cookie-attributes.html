Cookie Security Attributes

HttpOnly Server-side only

Set by server via Set-Cookie: token=abc; HttpOnly

JavaScript cannot read HttpOnly cookies via document.cookie. This protects session tokens from XSS attacks.

Cannot be demonstrated in client-side JS — by design, the browser hides these cookies from JavaScript.

Secure HTTPS only

A cookie with the Secure flag is only transmitted over encrypted (HTTPS) connections. Prevents interception over HTTP.
