When to use each

localStorage User preferences (theme, language), non-sensitive tokens, UI state that should survive navigation and tab close.
sessionStorage Multi-step form data, wizard progress, tab-specific state you want discarded when the tab closes.
Cookies Session tokens (HttpOnly + Secure), authentication state that the server needs to read, cross-subdomain sharing.
IndexedDB Large datasets, offline-first apps, structured data that needs querying, file/blob caching, anything over 5 MB.
